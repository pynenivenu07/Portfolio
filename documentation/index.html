Customization

Logo

The template needs two logo files to be set.

The /images/logo/logo.png is displayed when we see the top of the page.
The /images/logo/logo-dark.png is displayed on white background as the page scrolls down.

Logo appropriate size should be 150x40px

 

Images

All images are in /images folder in the main template folder.

All images are replaced with placeholders with the appropriate resolution written on it like about.jpg - have resolution of 150x150 pixels.

Images folder structure:

/images/backgrounds/ - contain all background images.
/images/logo/ - contain images of logo for the website.
/img/blog/ - contain images for the blog section and pages.
/img/portfolio/ - contain images for the portfolio section.

Contact Form Setup

Form is working on mail.php script on the root directory.

Setup is very simple, just change below line in mail.php file to your email address.

// Set the recipient email address.
// FIXME: Update this to your desired email address.
$recipient = "example@example.com";
Please remember! The script will not run locally, it must run on the server.
